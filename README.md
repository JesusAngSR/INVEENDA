# Inveenda - Sistema de Inventario y Tienda Online

esta aplicación web desarrollada con Node.js, Express y MongoDB, permite gestionar productos desde un dashboard administrativo y mostrarlos en una tienda online dinámica.

permitiendo acceso al dashboard solo a usuarios administradores.

---

## Instalación y ejecución

### 1. Clonar el repositorio

```bash
git clone https://github.com/jesusAngSR/INVEENDA
```

### 2. Instalar dependencias

```bash
npm install
```

### 3. Mongo DataBase

Instala MongoDB en tu computadora y asegúrate de que el servicio esté activo.
ya sea desde su interfaz de consola o Mongo Compass su interfaz grafica.

### 4. Ejecutar el proyecto

Ejecutar el proyecto con:

```bash
npm start
```

o con el modo desarrollo:

```bash
npm run dev
```

### 5. Abrir en el navegador

entra a tu navegador y escribe:

```bash
http://localhost:3000
```
